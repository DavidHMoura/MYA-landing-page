💻 Sobre o Projeto
Este projeto foi concebido para ser uma interface rápida e responsiva. Como desenvolvedor com foco em Back-end, priorizei a organização do código e a eficiência do bundle, garantindo que o carregamento seja instantâneo e a manutenção simplificada.

Principais Funcionalidades:
Layout Totalmente Responsivo (Mobile-First).

Componentes Reutilizáveis com shadcn/ui.

Otimização de ativos e imagens via Vite.

Navegação suave e performance otimizada (Lighthouse Core Web Vitals).

🛠 Tecnologias
O ecossistema do projeto baseia-se em ferramentas modernas que garantem segurança no desenvolvimento:

Framework: React 18

Linguagem: TypeScript (Tipagem estrita para evitar comportamentos inesperados)

Estilização: Tailwind CSS & Lucide React (Ícones)

Build Tool: Vite

UI Components: shadcn/ui

🛡️ Arquitetura e Segurança
Fiel à minha paixão por Cibersegurança, o projeto segue diretrizes para manter a integridade do código:

Dependency Audit: Verificação constante de vulnerabilidades em pacotes NPM.

Environment Isolation: Configuração pronta para variáveis de ambiente protegidas.

Static Analysis: Uso de ESLint e Prettier para garantir padronização e evitar débitos técnicos.

⚙️ Instalação e Uso
Para rodar o projeto localmente em sua máquina (preferencialmente em um ambiente Linux 🐧), siga os passos abaixo:

Bash
# 1. Clone o repositório
git clone https://github.com/davidhmoura/MYA-landing-page.git

# 2. Acesse o diretório
cd MYA-landing-page

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
O servidor será iniciado em http://localhost:5173.

🚀 Deploy
O deploy é realizado automaticamente via GitHub Pages. Para gerar a build de produção manualmente:

Bash
npm run build
Os arquivos otimizados serão gerados na pasta /dist.
